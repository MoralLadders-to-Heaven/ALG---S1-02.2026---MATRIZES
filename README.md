# Trabalho de Matrizes em Linguagem C

## Visão Geral

Este repositório reúne a solução do trabalho de Algoritmos em C sobre matrizes, com foco na aplicação prática de estruturas bidimensionais, laços de repetição e processamento de valores armazenados em memória.

O programa foi estruturado com um menu interativo que permite executar cada questão separadamente, mantendo a organização do código e facilitando a leitura dos resultados.

---

## Objetivo

O objetivo deste trabalho é desenvolver programas em linguagem C capazes de:

- manipular matrizes bidimensionais
- realizar operações entre elementos de uma matriz
- identificar valores específicos dentro de uma estrutura
- exibir os resultados de forma clara e organizada

Além disso, o projeto reforça a prática de programação estruturada e a construção de soluções separadas por funcionalidade.

---

## Funcionalidades Desenvolvidas

O sistema apresenta um menu principal com as seguintes opções:

- executar a Questão 1
- executar a Questão 2
- sair do programa

### Questão 1 — Soma de Matrizes

O programa lê duas matrizes `A` e `B`, ambas de tamanho `3 x 3`, e calcula uma terceira matriz `C`, na qual cada posição corresponde à soma dos elementos equivalentes das duas matrizes de entrada.

Ao final, são exibidas:

- a matriz `A`
- a matriz `B`
- a matriz resultante `C`

### Questão 2 — Maior Valor em uma Matriz

O programa lê uma matriz de inteiros de tamanho `4 x 4`, identifica o maior valor armazenado e informa sua posição exata dentro da matriz.

Ao final, são exibidos:

- a matriz informada
- o maior valor encontrado
- a linha e a coluna onde esse valor está localizado
